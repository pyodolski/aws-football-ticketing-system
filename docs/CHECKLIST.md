# 🎯 부하 테스트 및 보고서 작성 체크리스트

## ⏰ 테스트 30분 전 준비

### 1. Cloud SQL 시작

- [ ] GCP Console → SQL → 인스턴스 선택 → **시작** 클릭
- [ ] 5-10분 대기 (상태: 실행 중)

### 2. 인스턴스 그룹 시작

- [ ] Compute Engine → 인스턴스 그룹 → `football-app-group-v2` → **수정**
- [ ] 최소 인스턴스: `2`, 최대 인스턴스: `5`
- [ ] **저장** 클릭
- [ ] 10분 대기 (인스턴스 2개 생성 확인)

### 3. 서비스 접속 확인

- [ ] 브라우저에서 `http://136.110.134.8/` 접속
- [ ] 로그인 페이지 정상 표시 확인
- [ ] 회원가입 테스트
- [ ] 로그인 테스트
- [ ] 경기 목록 조회 확인
